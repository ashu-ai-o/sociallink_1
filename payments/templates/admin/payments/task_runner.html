📧 Send Payment Receipt Email

This task requires a specific Payment ID — run it from the Payments list using the admin action.

1
Go to the Payments list and find a payment with status = success.
2
Tick the checkbox next to one or more payments.
3
In the Action dropdown at the top, select "📧 Send payment receipt email (via Celery)" then click Go.
4
Check your Celery worker terminal — you will see Receipt email sent for payment <uuid>.
→ Go to Payments List

📌 Before You Start

🖥️

Start the Worker

Open a terminal and run:
celery -A ai_platform worker -l info
This is where task output will appear.

▶️

Run a Task

Click Run Now on any task above. A confirm dialog will appear. Click OK to queue the task instantly.

👀

Watch the Output

Switch to your worker terminal. You will see the task name, logs, and return value printed in real-time.

⚠️

Caution Tasks

Red-bordered tasks (like Reset Daily Tokens) modify live data for all users. Only run these intentionally.
